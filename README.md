# Quadratic-Regression-application
Quadratic regression analysis to determine optimal inventory levels for Turkish dairy and meat companies (Pınar Süt &amp; Pınar Et) using Python.
## Context
Academic project — Master en Gestion Financière

## Objective
Identify key variables affecting ROA (Return on Assets) 
for Turkish dairy and meat companies using quadratic 
regression modeling in Python.

## Key findings

A variable was considered impactful on ROA only when 
the quadratic regression produced an inverted U-shape — 
indicating a clear optimal point beyond which returns diminish.
Variables without this shape were deemed statistically 
insignificant drivers of profitability.

## Tools
Python · NumPy · Pandas · Matplotlib · Scikit-learn
